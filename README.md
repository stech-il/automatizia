# Automatizia – שרת צ'אט וואטסאפ

שרת מרכזי שמחבר צ'אט באתר לוואטסאפ של מנהל האתר.

## פריסה ב-Render (Blueprint)

1. חבר את הריפו [stech-il/automatizia](https://github.com/stech-il/automatizia) ל-[Render](https://render.com)
2. בחר **New > Blueprint** – Render יזהה את `render.yaml` אוטומטית
3. הגדר משתני סביבה:
   - `ADMIN_USER` – שם משתמש
   - `ADMIN_PASS` – סיסמה
4. פרוס
5. גלוש ל-`https://your-app.onrender.com/admin`
6. סרוק QR והגדר אתרים

## הרצה מקומית

```bash
npm install
npm start
```
